# danielmatei.dev

Personal developer portfolio built with React, TypeScript, and Vite.

## Commands

- `npm run dev` starts the local development server.
- `npm run build` type-checks and creates a production build.
- `npm run lint` checks the source with Oxlint.
- `npm run preview` serves the production build locally.

## Structure

```text
src/
  components/  Reusable UI components
  styles/      Global styles, design tokens, and app styles
  App.tsx      Application shell
  main.tsx     Browser entry point
```

The production build is configured for Firebase Hosting.
